Design a grid, fill in clues, and export a JSON you can drop straight into the crossword player. ← Back to puzzles

💡 How to use this editor

  1. Set a Title and the number of Rows / Columns, then click New grid.
  2. Click a cell to select it (click again to switch between Across / Down).
  3. Type A–Z to fill letters. The cursor auto-advances in the current direction.
  4. Make a black square: double-click a cell, or press . / #. Do it again to turn it white.
  5. Backspace clears a letter · Space flips Across/Down · Arrow keys move.
  6. Numbers appear automatically from your black-square pattern — you don't place them by hand.
  7. Fill each Across and Down clue on the right. The answer beside each box is read live from your letters.
  8. Click Export & open in player to preview it instantly in the crossword player, or use Download .json / Copy to save the file.
Tip: keep Rotational symmetry on to mirror black squares automatically (like real crosswords). You can also Load JSON to edit an existing puzzle.
